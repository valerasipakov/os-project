Набор bash-скриптов для анализа результатов тестов по заданной группе, предмету и тесту. Скрипт выводит:

имя(имена) студента(ов) с максимальным числом правильных ответов и значение;

имя(имена) студента(ов) с максимальным числом неправильных ответов и значение.

Для предметов зашито количество вопросов:

Цирковое_Дело — 25

Поп-Культуроведение — 5

Требования

bash ≥ 4

Unix-подобная ОС (Linux/macOS)

Быстрый старт
chmod +x scripts/main.sh scripts/lib/*.sh
./scripts/main.sh --group=Ae-21-22 --subject="Цирковое_Дело" --test=TEST-1 --action=both


Без флагов скрипт спросит значения интерактивно (группа → предмет → тест → что вывести).

Запуск и флаги
./scripts/main.sh [--group=<строка>] [--subject=<строка>] [--test=<имя>] [--action=<режим>]

Флаг	Пример	Обязательно	Назначение
--group	--group=Ae-21-22	нет	Номер группы. Если не задан — будет запрошен.
--subject	--subject="Цирковое_Дело"	нет	Предмет: Цирковое_Дело или Поп-Культуроведение.
--test	--test=TEST-1	нет	Имя файла теста (TEST-1 … TEST-4).
--action	--action=both	нет	Режим вывода: max-correct, max-wrong, both (по умолчанию).
Примеры

Максимум правильных:

./scripts/main.sh --group=Ae-21-22 --subject="Поп-Культуроведение" --test=TEST-2 --action=max-correct


Максимум неправильных:

./scripts/main.sh --group=Ae-21-22 --subject="Цирковое_Дело" --test=TEST-3 --action=max-wrong


Оба показателя:

./scripts/main.sh --group=Ae-21-22 --subject="Цирковое_Дело" --test=TEST-1

Переменные окружения

DATA_ROOT — корень каталога с данными (по умолчанию ./data/labfiles-25):

DATA_ROOT=/abs/path/to/labfiles-25 ./scripts/main.sh --group=Ae-21-22 --subject="Цирковое_Дело" --test=TEST-1

Формат данных

В файлах тестов строки вида:

<Группа>;<ФамилияИО>;<Дата>;<Правильных>;<Оценка>


Неправильных = <Всего_вопросов_по_предмету> - <Правильных>.

Вывод

Пример:

Группа: Ae-21-22
Предмет: Цирковое_Дело
Тест: TEST-1
Макс правильных (18): IvanovII PetrovPP
Макс неправильных (7): SidorovSS


Если у нескольких студентов значение совпадает, выводятся все.
